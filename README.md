## Overview

This repository demonstrates GitHub Actions and CI/CD automation with 4 workflows.

## Workflows

### 1. CI Pipeline (ci.yml)
- Linting job
- Testing job
- Build job

### 2. Advanced CI (advanced-ci.yml)
- Setup and test
- Code quality checks

### 3. Security Audit (security.yml)
- Daily security checks

### 4. Deploy (deploy.yml)
- Deployment workflow

## Quick Start

```bash
npm install
npm test
npm run lint
npm run build
```
